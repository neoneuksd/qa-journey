# API and JSON Basics

## API
API is a way for the client to communicate with the server.

## HTTP
HTTP is the set of rules for communication between client and server.

## Request
A request is what the client sends to the server.

## Response
A response is the server's answer to the client's request.

## JSON
JSON is a data format in which the server often sends information to the client.

## Key
A key is the name of a field in JSON.

## Value
A value is the content of a field in JSON.

## Object
An object is a structure of key-value pairs written in curly braces {}.

## Array
An array is a list of values written in square brackets [].

## Endpoint
An endpoint is a specific address the client uses to access the API.

## Params
Params are request parameters that уточняют which data should be returned.

## Header
A header is service information about the request.

## Body
A body is the main content of the request, the data the client sends to the server.
